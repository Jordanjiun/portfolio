Group Design Project

For my final year of university, I was the group leader of the MORPHEUS (Modular Operational Remote Package Handling Electric UAV System) Group Design Project (GDP) in which our team attempted to design and build a drone that is capable of autonomously detecting, tracking, collecting, and depositing a package on its own based on computer vision. While managing the project, I specialised in the avionics and software components of the system which meant that I had a lot of fun learning about Python's OpenCV and how to use fiducial markers to program and test a control algorithm with a bunch of simulations and eventual in-flight tests. You can check out the code we worked on here or if you would like to find out more about the overall project, feel free to read our final report.
